## 🔧 Views PHP Warnings Fix (Drupal 11)

**Problem**: 18 PHP warnings på front page: `Undefined array key "element_default_classes"`

**Root cause**: Drupal 11 kräver explicit `element_default_classes` key på alla Views fields

**Lösning**:
1. Fixade 6 views configs:
   - hero_media.yml (3 fields)
   - featured_products.yml (3 fields)
   - browse_by_application.yml (1 field)
   - performance_features.yml (numeric filter "in" key)
   - media_library.yml (2 fields)
   - watchdog.yml (1 field)

2. La till `element_default_classes: true` efter varje `element_label_type`

3. Rensade Twig cache:
```bash
ddev exec rm -rf /var/www/html/web/sites/default/files/php
ddev drush sqlq "TRUNCATE cache_render"
ddev drush cr
```

**Resultat**: ✅ 0 PHP warnings synliga, inga nya fel i watchdog

**Dokumentation**: `/docs/03-solutions/views/VIEWS-FIELD-CONFIGURATION.md`
